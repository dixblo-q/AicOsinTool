## 🛠️ Özellikler

- **Social-Account OSINT:** Kullanıcı adını tüm popüler platformlarda arar.
- **Phone-Number OSINT:** Operatör, bölge ve geçerlilik kontrolü.
- **Domain/IP OSINT:** Whois, DNS ve IP lokasyon verileri.
- **E-Mail OSINT:** Veri sızıntısı (breach) kontrolleri.
- **Person OSINT:** İsim üzerinden gelişmiş Google Dorks sorguları.
- **Port Scanner:** Hedef sistemdeki açık portları tespit eder.
- **Metadata OSINT:** Dosyaların içine gizlenmiş verileri analiz eder.
- **Database GUI:** Tüm tarama geçmişini şık bir arayüzde saklar.
